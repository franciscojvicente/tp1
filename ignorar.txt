git ignore sirve para ignorar archivos o carpetas de un nombre o tipo especifico en un proyecto 

Para crear un archivo .gitignore local, crea un archivo de texto y 
asígnale el nombre ".gitignore" (recuerda incluir el . al principio). 
Luego, edita este archivo según sea necesario. 
Cada nueva línea debe incluir un archivo o carpeta adicional que quieras 
que Git lo ignore.
